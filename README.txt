# Surabhi Feeds Website — Image Setup Guide

## 📁 Folder Structure

```
surabhi-feeds-website/
├── index.html
├── products.html
├── style.css
├── products.css
├── script.js
└── images/
    └── products/
        ├── hipro.png          ← Surabhi Hi-Pro bag photo
        ├── suraksha-plus.png  ← Suraksha Plus bag photo
        ├── gold-pellet.png    ← Gold Pellet bag photo
        ├── pragathi.png       ← Pragathi Pellet bag photo
        └── super.png          ← Super Pellet bag photo
```

## 🖼️ How to Add Your Product Images

1. Take clear photos of each product bag (preferably on a white or transparent background)
2. Rename them exactly as shown above (case sensitive!)
3. Place them inside the `images/products/` folder
4. Recommended image size: **600×700px or larger**, PNG format (transparent background looks best)

## ✅ Tips for Best Results

- **PNG with transparent background** looks most professional on the colored panels
- Use **natural lighting** or a plain white background when photographing bags
- If images are JPG, rename them to `.jpg` and update the `src` in products.html accordingly
- Minimum image width: **400px**
